Quick Start
Get running in under 2 minutes

🖥️ Windows / Linux / macOS

  1. Download ascii_library_tool.py
  2. Optionally download Database1.accdb for Access mode
  3. Run: python ascii_library_tool.py

📱 Android via Termux

  1. Download ascii_library_tool.py → /sdcard/Download
  2. Download run_termux.py or run_termux.sh
  3. In Termux, execute the launcher script
